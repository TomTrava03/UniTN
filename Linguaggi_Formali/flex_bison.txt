    Syntax:
    %{ code
    }%

    shorthands for patterns

    %%
    pattern-1 {action-1};
    pattern-2 {action-2};
    ...

    %%
    user routines, copied into lex.yy.c

    Metacharacter matches:
    .   Any character (except newline)
    \n  newline
    *   zero(or more)
    +   one(or more)
    ?   zero or one
    []  character class
    ^   beginning of line (negation if used in [])
    $   end of line
    a|b a or b
    ()  grouping
    "+" literal "+"
    {}  regexp defined in the preamble
